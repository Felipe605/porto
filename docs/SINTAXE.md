# Sintaxe do Porto

## Tipos de Dados

### Números
```porto
variavel inteiro = 42
variavel decimal = 3.14
variavel negativo = -10
```

### Texto
```porto
variavel nome = "Mista"
variavel frase = "Porto é incrível!"
```

## Operadores

### Aritméticos
- `+` Adição
- `-` Subtração
- `*` Multiplicação
- `/` Divisão
- `%` Módulo (resto da divisão)

### Comparação
- `==` Igual
- `!=` Diferente
- `>` Maior que
- `<` Menor que
- `>=` Maior ou igual
- `<=` Menor ou igual

## Estruturas de Controle

### Condicional
```porto
se condicao entao
    # código
fim
```

### Loop
```porto
enquanto condicao faca
    # código
fim
```

## Entrada/Saída

### Saída
```porto
escreva "mensagem"
escreva variavel
```

### Entrada
```porto
leia variavel
```
